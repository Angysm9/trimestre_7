##  Objetivo del repositorio 
* Este repositorio tiene como objetivo guardar, organizar y crear diversos trabajos y códigos artísticos y académicos realizados en Visual Studio Code (VSC), utilizando distintos lenguajes de programación como HTML, CSS, JavaScript (JS) y JSON. Su función principal es servirme como un archivo de seguimiento del curso, donde se pueda acceder fácilmente (yo, el profesor y terceras personas) al contenido para consultar o re-utilizar dichos materiales.
* La mayoría de estos ejercicios y proyectos se desarrollan en clase y posteriormente son revisados y entregados al profesor del curso para su evaluación. El editor de código VSC, junto con GitHub, me permite mantener al tanto a mi Profesor sobre las actividades realizadas y, al mismo tiempo, llevar un seguimiento claro de cada una de ellas. Además, este README me funciona como un registro de mi proceso de aprendizaje, permitiéndome observar mis avances técnicos e identificar cosas que me hacen falta mejorar. También facilita la colaboración, el intercambio de ideas y como ya lo he planteado, me permite mejorar los trabajos y códigos realizados.
